# キャンプギアリスト

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - キャンプギアの登録・検索・集計ができます。（集計は実装中）

  - 検索はアイテム名やメーカーを入力して検索。集計は現状すべてのアイテムの合計金額、総重量が出せています。
